### SLIDE 6 — BUSINESS MODEL
**Heading:** Three tiers. Near-zero marginal cost on the free tier.
**3 tier cards:**

**Free**
No account required for basic tools. 3 uses/day enforced by browser cookie. Supported by display advertising. Marginal cost per user: fractions of a cent.

**Pro — $9/month or $79/year**
Unlimited AI tool uses. Batch processing. 2GB file limit. API costs per active Pro user: ~$1.50–$2.50/month. Margin: strong.

**Business — Consultation pricing**
Team access, shared dashboard, priority processing, no ads. Scoped per organization. Includes access to the Proprietary Tool Program.
